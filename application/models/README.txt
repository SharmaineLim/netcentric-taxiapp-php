
Please SET UP YOUR DATABASE.
============================

STEPS:
------

1. Create the database using
     "application/models/database.sql"

2. Go to
     "application/config/database.php"

3. Set up the following:
     $db['default']['username'] = '<your mysql username>';
     $db['default']['password'] = '<your mysql password>';
     $db['default']['database'] = 'taxi_reporting_ph';
